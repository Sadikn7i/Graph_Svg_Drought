## About the Research

This study examines food security outcomes among **2,450 displaced households** in Somalia using nationally representative World Bank survey data (2020–2021). It investigates whether climate-, conflict-, or economically-driven displacement produces different food security outcomes under COVID-19 income shocks.

### Key Findings

| # | Finding |
|---|---------|
| 1 | **Displacement driver matters less than expected** — climate vs. conflict differences disappear in multivariate weighted models |
| 2 | **Protracted displacement (>12 months) is protective** — households displaced longer show lower food insecurity (OR = −1.063***) |
| 3 | **COVID-19 income shocks erase adaptation gains** — "other-reason" displaced households are hit hardest, yet remain outside standard aid frameworks |

### Food Insecurity Rates by Population Type

| Population | Food Insecurity Rate |
|------------|---------------------|
| Host Communities | 49.7% |
| IDP (Camp) | 76.2% |
| IDP (Outside Camp) | 60.7% |
| Refugees | 79.2% |
| Returnees | 69.3% |

## Usage

### View Locally
1. Clone or download this repository
2. Open `graph.html` in **Chrome or Edge**
3. The figure renders immediately in your browser

### Download the Figure
Three export options are available directly from the HTML page:

| Button | Format | Resolution | Best For |
|--------|--------|------------|----------|
| Download SVG | Vector | Infinite | Editing, print |
| Download PNG | Raster | 2656 × 1062 px (2×) | Submissions, presentations |
| Download JPG | Raster | 2656 × 1062 px (2×) | Web, email |

> PNG/JPG export requires **Chrome or Edge** — Safari may block canvas rendering due to security restrictions.
